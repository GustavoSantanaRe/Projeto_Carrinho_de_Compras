# 💻 Sobre o Projeto

O **Carrinho de Compras JS** é uma aplicação web interativa desenvolvida com foco em **lógica de programação** e **manipulação do DOM** utilizando JavaScript puro.

O projeto simula um sistema de carrinho de compras, onde o usuário pode visualizar produtos, adicioná-los ao carrinho e interagir dinamicamente com a interface.

Este projeto foi desenvolvido como parte do curso **Técnico em Desenvolvimento de Sistemas** da escola **SENAI A. Jacob Lafer**, com o objetivo de colocar em prática conceitos fundamentais do desenvolvimento Front-end.

---

# ⚙️ Funcionalidades e Conceitos Aplicados

Durante o desenvolvimento, foram aplicados os seguintes conceitos técnicos:

## 📌 Manipulação de DOM (JavaScript)
- Seleção de elementos com `querySelector` e `querySelectorAll`
- Atualização dinâmica de conteúdos na página
- Criação e remoção de elementos HTML via JavaScript

## 🧠 Lógica de Programação
- Estruturação de funções para controle do carrinho
- Manipulação de arrays para armazenar produtos
- Cálculo de valores (total da compra)

## 🖱️ Eventos JavaScript
- Uso de `addEventListener` para interações do usuário
- Eventos de clique para adicionar/remover itens

## 🧱 HTML Estruturado
- Organização da página com boas práticas
- Integração com o JavaScript para comportamento dinâmico

## 🎨 CSS
- Estilização pronta (fornecida)
- Layout responsivo e organizado

## 🔧 Versionamento
- Controle de versão utilizando **Git**
- Publicação do projeto no **GitHub**
